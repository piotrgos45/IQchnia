# IQchnia
Projekt na uczelnie, który ma za zadanie stworzenie Inteligentnej Ksiązki Kucharskiej

## Frontend

Uruchomienie 

Css: [Tailwind](https://tailwindcss.com/)

Biblioteka Komponentow: [MUI](https://mui.com/)

```bash
cd IQchnia
yarn
yarn dev
```
Budowanie

```bash
cd IQchnia
yarn build
```
